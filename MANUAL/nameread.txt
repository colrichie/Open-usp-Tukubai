nameread(USP)

＜名前＞
nameread : name 形式のファイルを読む (USP-TOOL)

＜書式＞
Usage   : nameread [-el] <name> <namefile>
Option  : -d<c>       空白を置換
        : -i<string>  ヌルデータの初期化
Version : Thu Dec 15 13:46:38 JST 2011

＜説明＞
name 形式ファイルから name を指定して値を読みます。
<namefile> が "-" あるいは指定が無いときは、
標準入力ファイルを取り込みます。
-l オプションで name も合わせて出力します。
-e オプションで正規表現を指定できます。
-d オプションで空白データを置換する文字を指定します。
(指定がない場合は空白データを削除します)
-i オプションでヌルデータの初期化を行います。 

＜例＞
$ cat namefile
A usp
B usp laboratory
C

# 通常の出力
$ nameread A namefile
usp

# name 形式のデータは空白を含む
$ nameread B namefile
usp laboratory

# 空白を変換したいとき
$ nameread -d_ B namefile
usp_laboratory

# 空白を削除したいとき
$ nameread -d B namefile
usplaboratory

# 無い値を読むと空行が出力
$ nameread C namefile    

# ヌルデータを初期化したいとき 
$ namread -i @@@ C namefile
@@@

# 無い名前を読むと何も出力しない(-i オプションも効かない)
$ nameread D namefile    
$

# ラベルと一緒に出力するとき
$ nameread -l A namefile
A usp

# 正規表現指定
$ cat namefile2
A01 1
A02 2
A03 3
A10 5
$ nameread -e '^A0[0-9]$' namefile2
1
2
3
$ nameread -el '^A0[0-9]$' namefile2
A01 1
A02 2
A03 3
